 MODS 评分

 
0 分值: ICU 死亡率 0%，住院死亡率 0%，入住 ICU 2 天
1 - 4 分值: ICU 死亡率 1-2%，住院死亡率 7%，入住 ICU 3 天
5 - 8 分值: ICU 死亡率 3-5%，住院死亡率 16%，入住 ICU 6 天
9 - 12 分值: ICU 死亡率 25%，住院死亡率 50%，入住 ICU 10 天
13 - 16 分值: ICU 死亡率 50%，住院死亡率 70%，入住 ICU 17 天
17 - 20 分值: ICU 死亡率 75%，住院死亡率 82%，入住 ICU 21 天
21 - 24 分值: ICU 死亡率 100%，住院死亡率 100%

 
 

 

 

 
使用的方程式
 
PaO2/FIO2 = 100 * PaO2 / FIO2
以血压校正的心率 = 心率 * 中心静脉压 / 平均动脉压
MODS评分 = PaO2/FIO2分值 + 血小板分值 + 胆红素分值 + 以血压校正的心率分值 + 肾功能分值 + gcs分值

 

 
计算详情和变量
  • PaO2/FIO2分值 是通过此有序逻辑分配的数值:
    • 如果 PaO2/FIO2 >300 PaO2/FIO2分值 = 0
    • 否则，如果 PaO2/FIO2 ≥226 PaO2/FIO2分值 = 1
    • 否则，如果 PaO2/FIO2 ≥151 PaO2/FIO2分值 = 2
    • 否则，如果 PaO2/FIO2 ≥76 PaO2/FIO2分值 = 3
    • 否则 PaO2/FIO2分值 = 4
  • 以血压校正的心率分值 是通过此有序逻辑分配的数值:
    • 如果 以血压校正的心率 ≤10 以血压校正的心率分值 = 0
    • 否则，如果 以血压校正的心率 ≤15 以血压校正的心率分值 = 1
    • 否则，如果 以血压校正的心率 ≤20 以血压校正的心率分值 = 2
    • 否则，如果 以血压校正的心率 ≤30 以血压校正的心率分值 = 3
    • 否则 以血压校正的心率分值 = 4
  • 血小板分值 是通过此有序逻辑分配的数值:
    • 如果 血小板 >120 x10(3)/mcL 血小板分值 = 0
    • 否则，如果 血小板 ≥81 x10(3)/mcL 血小板分值 = 1
    • 否则，如果 血小板 ≥51 x10(3)/mcL 血小板分值 = 2
    • 否则，如果 血小板 ≥21 x10(3)/mcL 血小板分值 = 3
    • 否则 血小板分值 = 4.
  • 胆红素分值 是通过此有序逻辑分配的数值:
    • 如果 胆红素 ≤20 mcmol/L 胆红素分值 = 0
    • 否则，如果 胆红素 <61 mcmol/L 胆红素分值 = 1
    • 否则，如果 胆红素 <121 mcmol/L 胆红素分值 = 2
    • 否则，如果 胆红素 ≤240 mcmol/L 胆红素分值 = 3
    • 否则 胆红素分值 = 4.
  • 肾功能分值 是通过此有序逻辑分配的数值:
    • 如果 肌酐 ≤100 mcmol/L 肾功能分值 = 0
    • 否则，如果 肌酐 <201 mcmol/L 肾功能分值 = 1
    • 否则，如果 肌酐 <351 mcmol/L 肾功能分值 = 2
    • 否则，如果 肌酐 ≤500 mcmol/L 肾功能分值 = 3
    • 否则 肾功能分值 = 4.
  • gcs分值 是通过此有序逻辑分配的数值:
    • 如果 Glasgow 昏迷评分 >14 points gcs分值 = 0
    • 否则，如果 Glasgow 昏迷评分 >12 points gcs分值 = 1
    • 否则，如果 Glasgow 昏迷评分 >9 points gcs分值 = 2
    • 否则，如果 Glasgow 昏迷评分 >6 points gcs分值 = 3
    • 否则 gcs分值 = 4.
