注意
  • 该计算器有助于预测以下动脉粥样硬化性心血管疾病事件的10年风险：
    • 非致命性心肌梗死的 首次发病
    • 冠心病（CHD）致死
    • 致命或非致命性中风
  • 该计算器可能会高估相关风险（参考Lancet文献），若有任何问题，需要与患者商讨。
  • 风险评估开发主要基于白种和非裔美国人受试群体。对于美洲印第安人、南亚血统的亚裔美国人以及波多黎各人而言， 相关风险也有可能被低估。对于东亚血统的亚裔美国人以及部分墨西哥裔美国人而言， 风险可能被高估。

 
使用的方程式
 
术语 = (C年龄 * ln(年龄)) + (CSq年龄 * sq(ln(年龄))) + (CTotalChol * ln(总胆固醇)) + (C年龄TotalChol * ln(年龄) * ln(总胆固醇)) + (CHDLChol * ln(高密度脂蛋白胆固醇)) + (C年龄HDLChol * ln(年龄) * ln(高密度脂蛋白胆固醇)) + (服用降血压药 * C服用降血压药s * ln(收缩期血压)) + (服用降血压药 * C年龄服用降血压药s * ln(年龄) * ln(收缩期血压)) + (!服用降血压药 * COffHypertensionMeds * ln(收缩期血压)) + (!服用降血压药 * C年龄OffHypertensionMeds * ln(年龄) * ln(收缩期血压)) + (C吸烟者 * 吸烟者) + (C年龄吸烟者 * ln(年龄) * 吸烟者) + (C糖尿病 * 糖尿病)
10年风险 = 100 * (1 - S10e(术语-Mean术语))
