注意
  • 在该计算器中，基于年龄和身高百分比确定受试者的血压百分位。
  • 第一期高血压是指，收缩压或舒张压介于第 95 百分位和第 99 百分位加上 5 毫米水银柱之间。
  • 第二期高血压是指，收缩压或舒张压超过第 99 百分位加上 5 毫米水银柱。
  • 血压超过 120/80 毫米水银柱的儿童和青少年也被视为高血压前期。
  • 在极端情况下（>97百分位数或<3百分位数），小的百分位数差异 表示有临床重要性的差异。在这些极端情况下，Z-评分能更准确地 反映测量值偏离平均值的程度，对于追踪变更是一个有用的工具。
  • 此计算器使用的 LMS 数据见以下数据表格.
  • 对于所有 LMS 表，表格中对应患者特定值的行见此表的第一 列；检索用于 Z-评分公式中 L（Box-Cox 幂）、M（中位数）和 S（广义 变异系数）的数值。
  • Z-评分可用于对 Z-评分正态表格百分位数 表中的对应百分位数进行表格搜索。表格左侧（Y 轴）的数字表示 Z 评分的个位和小数点后一 位数，而表格上方（X 轴）的数字表示 Z 评分小数点后两位数。

 
使用的方程式
 
年龄月 = 年龄 * 12
身高Z = ((身高/M)L - 1) / (L * S)
身高百分位 = Z对应百分位数(身高Z)
年龄系数 = 年龄 - 10
平均收缩压 = 102.01027 + (1.94397 * 年龄系数) + (0.00598 * 年龄系数2) - (0.00789 * 年龄系数3) - (0.00059 * 年龄系数4) + (2.03526 * 身高Z) + (0.02534 * 身高Z2) - (0.01884 * 身高Z3) + (0.00121 * 身高Z4)
平均舒张压 = 60.50510 + (1.01301 * 年龄系数) + (0.01157 * 年龄系数2) + (0.00424 * 年龄系数3) - (0.00137 * 年龄系数4) + (1.16641 * 身高Z) + (0.12795 * 身高Z2) - (0.03869 * 身高Z3) - (0.00079 * 身高Z4)
收缩压Z = (收缩压 - 平均收缩压) / 10.4855
舒张压Z = (舒张压 - 平均舒张压) / 10.9573
收缩压百分位 = Z对应百分位数(收缩压Z)
舒张压百分位 = Z对应百分位数(舒张压Z)
收缩压阈值 = (2.326348 * 10.4855) + 平均收缩压 + 5
舒张压阈值 = (2.326348 * 10.9573) + 平均舒张压 + 5
