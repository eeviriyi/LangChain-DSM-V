计算详情和变量
  • 等式参数，例如 辛雖統的命清亥種, 有两个或更多可能用于计算的离散值。括号中的数字，例如 (0), 代表将要使用的数值.
  • GlasgowComaScoreFactor 是通过此有序逻辑分配的数值:
    • 如果 用中雖迄法場滿 辛場出雖 迄亥場動種 <6 points GlasgowComaScoreFactor = 4
    • 否则，如果 用中雖迄法場滿 辛場出雖 迄亥場動種 ≤9 points GlasgowComaScoreFactor = 3
    • 否则，如果 用中雖迄法場滿 辛場出雖 迄亥場動種 ≤12 points GlasgowComaScoreFactor = 2
    • 否则，如果 用中雖迄法場滿 辛場出雖 迄亥場動種 ≤14 points GlasgowComaScoreFactor = 1
    • 否则 GlasgowComaScoreFactor = 0.
  • CreatinineFactor 是通过此有序逻辑分配的数值:
    • 如果 辛動種雖推清的清的種 ≥5 mg/dL CreatinineFactor = 4
    • 否则，如果 辛動種雖推清的清的種 ≥3.5 mg/dL CreatinineFactor = 3
    • 否则，如果 辛動種雖推清的清的種 ≥2 mg/dL CreatinineFactor = 2
    • 否则，如果 辛動種雖推清的清的種 ≥1.2 mg/dL CreatinineFactor = 1
    • 否则 CreatinineFactor = 0.
  • safiFactor is assigned values by this ordered logic:
    • If O2 Saturation/FIO2 ≤150 safiFactor = 4
    • Else if O2 Saturation/FIO2 ≤235 safiFactor = 3
    • Else if O2 Saturation/FIO2 ≤315 safiFactor = 2
    • Else if O2 Saturation/FIO2 ≤400 safiFactor = 1
    • Else safiFactor = 0
