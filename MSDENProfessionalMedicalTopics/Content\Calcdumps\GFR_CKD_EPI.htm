Notes
  • GFR is estimated by an equation developed by the Chronic Kidney Disease Epidemiology Collaboration.
  • GFR = 141 * min(Serum_creatinine/kappa, 1)alpha * max(Serum_creatinine/kappa, 1) -1.209 * 0.993Age * Sex * Race
  • For females, the following values are used: Sex = 1.018; alpha = -0.329; kappa = 0.7
  • For males, the following values are used: Sex = 1; alpha = -0.411; kappa = 0.9

 
Equations used
 
GFR = 141 * Math.min(Serumcreatinine/kappa1),alpha * Math.max(Serumcreatinine/kappa1),-1.209 * 0.993Age * Sex * Race

 

 
Calculation Details and Variables
  • Equation parameters such as Race, have two or more discrete values that may be used in the calculation. The numbers in the parentheses, e.g. (1), represent the values that will be used.
