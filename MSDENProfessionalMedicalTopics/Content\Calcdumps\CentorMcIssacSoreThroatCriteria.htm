 Sore Throat Score

 
0 points: 2-3% chance of strep; no culture or antibiotics
1 point: 4-6% chance of strep; no culture or antibiotics
2 points: 10-12% chance of strep; culture and treat if positive
3 points: 27-28% chance of strep; culture and treat if positive
4 to 5 points: 38-63% chance of strep; culture and treat with antibiotic on clinical grounds

 
 

 

 
Equations used
 
Score = AgeFactor + TemperatureFactor + Cough + Tender Anterior Cervical Nodes + Tonsillar Swelling or Exudate

 

 
Calculation Details and Variables
  • AgeFactor is assigned values by this ordered logic:
    • If Age ≥45 yr AgeFactor = -1
    • Else if Age ≥15 yr AgeFactor = 0
    • Else AgeFactor = 1.
  • TemperatureFactor is assigned values by this ordered logic:
    • If Temperature >38 degC TemperatureFactor = 1
    • Else TemperatureFactor = 0.
  • Equation parameters such as Cough, have two or more discrete values that may be used in the calculation. The numbers in the parentheses, e.g. (0), represent the values that will be used.
